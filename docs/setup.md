# Setup Guide

## Prerequisites

- Python 3.9+
- Google Cloud Platform account
- gcloud CLI installed

## Installation

1. Clone the repository
2. Create virtual environment
3. Install dependencies

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
make install-dev
```
